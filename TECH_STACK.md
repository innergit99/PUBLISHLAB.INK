# Artisan AI: Technical Specification

## Core Architecture
- **Framework**: React 19 (Vite)
- **Language**: TypeScript 5.x
- **Styling**: Tailwind CSS 4.x Industrial Mode (Custom Palette: Indigo / Slate / Emerald)
- **Animations**: Framer Motion 12 (AnimatePresence for routing)
- **Video Engine**: Remotion 4.0 (Programmatic MP4 synthesis)
- **Visuals**: P5.js (Algorithmic backgrounds & neural net visualizations)

## Backend & AI
- **LLM Engine**: Google Gemini 1.5 Pro (via @google/genai)
- **Image Generation**: industrial-grade procedural logic + Gemini Image Service
- **State Management**: React Hooks + Local Storage Persistence
- **Database (Projected)**: Supabase / Pinecone (Vector Memory for Character Bibles)

## Industrial Standards
- **Design Pattern**: Glassmorphism / Holographic Dark Mode
- **Typography**: Inter (UI), Outfit (Headers), JetBrains Mono (Technical Details)
- **Asset Export**: 300 DPI PDF (jsPDF), High-res PNG (html2canvas)

## Infrastructure
- **Deployment**: Vercel / Modal (Serverless orchestration)
- **Security**: Sonatype Audit compliant dependency management
- **Testing**: TestSprite (Visual Regression), Vitest (Logic)
